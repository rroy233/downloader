# Downloader

一个简单的HTTP多线程(协程)下载器



## 环境依赖

* go 1.17



## 下载

前往[Releases](https://github.com/rroy233/downloader/releases)下载可执行文件



## 自行编译

```shell
go build -o downloader
```

或

```she
# 交叉编译
make all
```



## 使用

Mac or Linux

```shell
cd path/to/executable
./downloader
```

Windows

```
双击可执行文件
```
